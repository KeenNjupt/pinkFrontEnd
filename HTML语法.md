4. HTML常用标签
4.1 标题标签
<h1> - <h6> 表示一级标题到六级标题,一个标题独占一行
<h1>我是一级标题</h1>
h表示head
标签语义：设置标题
4.2 段落标签
<p>段落1</p>
p表示paragraph，不使用段落标签时，代码中的换行和多个空格只会转换为1个空格
标签语义：将HTML文档分割为多个段落
在一个段落中的文本会根据浏览器窗口的大小自动换行
段落和段落之间会有空隙
4.3 换行标签
段落中的文字从左到右依次排列，直到浏览器窗口的右端，才会自动换行。如果希望强制换行，使用换行标签<br />
br表示break
标签语义：强制换行
<br />为单标签
<br />标签只是简单地开始新的一行，跟段落不一样，段落之间会插入一些垂直的间距
4.4 文本格式化标签
加粗：<strong></strong> 或<b></b> 推荐<strong>
倾斜：<em></em> 或<i></i> 推荐<em>
删除线：<del></del> 或<s></s> 推荐<del>
下划线：<ins></ins> 或<u></u> 推荐<ins>
4.5 <div>和<span>标签
<div> 和 <span> 没有语义，他们就是一个盒子，用来装内容
div表示division，一行只能放一个div
span意为跨度，一行可以有多个span
4.6 图像标签和路径
<img>标签用于定义HTML页面中的图像
<img src="图像URL"/>
img 表示 image
src是<img>标签的必须属性，它用于指定图像文件的路径和文件名
所谓属性：简单理解就是属于这个图像标签的特性
alt属性 表示图像不能显示时，页面显示的文本，alternative
title属性：鼠标放在图像上，显示的文字
width 设置图像的宽度 只改宽度或高度时，另一个值会等比例放大或缩小
height 设置图像的高度
border 设置图像的边框粗细

注意：
1.图像标签的属性必须在标签名后面即 img后面
2.属性不分先后顺序，标签名于属性 属性与属性之间均以空格分开
3.属性采取键值对的方式，即key="value"的格式，属性="属性值"

文件路径之相对路径
相对路径：以引用文件所在位置为参考基础，而建立出的目录路径
简单来说，图片相对于HTML页面的位置
同级路径：图像位于HTML文件同一级 <img src="baidu.gif"/>
下一级路径：图像位于HTML文件下一级 <img src="imgages/baidu.gif"/>
上一级路径：图像位于HTML文件上一级 <img src="../baidu.gif"/>

文件路径之绝对路径
绝对路径：指目录下的绝对位置，直接到达目标位置，通常是从盘符开始的路径
例如：E:\Code\Code_FrontEnd\pinkFrontEnd\HTML第一天\案例\images\jack.jpg
或 完整的网络地址：https://i0.hdslb.com/bfs/static/jinkela/long/images/512.png
4.7 超链接标签
语法格式
<a href="跳转目标" target="目标窗口的弹出方式">文本或图像</a>

页面上显示的 "文本或图像"可以用来点击 跳转到目标url
a:anchor
href属性：指定链接目标的url地址，必须属性，当为标签应用href属性时，它就具有了超链接功能
target属性：指定链接页面的打开方式，_self为默认值 指当前标签页打开，_blank为在新标签页中打开
链接分类：
1. 外部链接，如 <a href="http://www.baidu.com" target="_blank">百度</a>
2. 内部链接：网站内部页面之间的相互链接，直接链接内部页面名称即可（href填写内部页面文件的路径），如<a href="index.html">首页</a>
3. 空链接：点击无效果，如：<a href="#">公司地址</a>
4. 下载链接：如果href里面地址是一个文件或压缩包，会下载这个文件 href填写文件路径 <a href="jack.zip">下载文件</a>
5. 网页元素的链接：在网页中的各种元素，文本，图像，表格，音频，视频等都可以添加超链接，在元素外部包裹<a></a>标签
   如 <a href="http://www.baidu.com"><img src="images/jack.jpg"/></a> 点击图片跳转至目标页面
6. 锚点链接：点击链接，可以快速定位到页面中的某个位置
   链接文本的href属性中设置值为#名字，如<a href="#two">第二集</a>
   在目标位置的标签上添加一个id属性，属性值为该名字，如<h3 id="two">第2集介绍</h3>

6.表格标签
<table>
   <tr> <td>单元格中的文字</td> ...</tr>
   ...
</table>

<table></table>用于定义表格的标签
<tr></tr>用于定义表格行的标签，必须嵌套在<table></table>标签中
<td></td>用于定义单元格,必须嵌套在<tr></tr>标签中
td指 table data
<th></th>用于定义表头单元格，其内容会加粗居中显示,嵌套在<tr></tr>标签中
th指 tablehead

table标签的属性
align属性，属性值: center left right，规定表格相对周围元素的对齐方式
border属性，属性值：1表示有边框，""为默认值表示无边框
cellpadding，属性值：像素值，表示单元格内部内容到边沿的空白距离，默认1像素
cellspacing，属性值：像素值，表示单元格之间的空白距离，默认2像素
width，属性值：像素值或百分比，表示表格的宽度

表格结构标签
<thead></thead> 用于定义表格的头部区域，<thead> 在 <table>标签内，内部包含<tr>
<tbody>></tbody> 用于定义表格的主体，<thead> 在 <table>标签内，内部包含<tr>

合并单元格
跨行合并：rowspan="合并单元格的个数"，合并后的目标单元格为上侧的单元格
跨列合并：colspan="合并单元格的个数"，合并后的目标单元格为左侧的单元格
三步曲：
1.选确定是跨行还是跨列
2.找到目标单元格。写上 合并方式=合并的单元格数量，如<td colspan= "2"></td>
3.删除多余的单元格

7.列表标签
7.1无序列表
<ul>标签表示无序列表，一般会以项目符号呈现列表项，列表项使用<li>标签定义
基本语法格式如下
<ul>
   <li>列表项1</li>
   <li>列表项2</li>
   <li>列表项3</li>
   <li>列表项4</li>
</ul>
1.无序列表各列表项之间没有顺序级别之分，是并列的
2.<ul></ul>中只能嵌套<li></li>,放入其他标签或文字是不允许的
3.<li></li>中可以放置任何内容
4.无序列表会带有自己的样式属性，实际使用中会用CSS来设置

7.2有序列表
<ol>标签表示有序列表，列表顺序以数字来显示，使用<li>标签来定义列表项
<ol>
   <li>列表项1</li>
   <li>列表项2</li>
</ol>
1.同<ul> <ol></ol>中只能嵌套<li></li>,放入其他标签或文字是不允许的

7.3自定义列表
自定义列表使用场景：常用于对术语或名词进行解释和描述，定义列表的列表项前没有任何项目符号
<dl>
   <dt>名词1</dt>
   <dd>名词1解释1</dd>
   <dd>名词1解释2</dd>
</dl>
<dl>标签表示自定义列表
<dt>标签表示名字
<dd>标签表示名字描述
1.<dl></dl>中只能包含<dt>和<dd>
2.<dt>和<dd>没有个数限制，经常是一个<dt>对应多个<dd>
